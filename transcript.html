How to Add Transcripts:

To add English and Turkish transcripts for community calls:

  1. Create a folder named transcripts in your project directory
  2. Inside it, create two subfolders: ENG and TR
  3. Add transcript files in the format: call-[number].txt
  4. For example: transcripts/ENG/call-1.txt and transcripts/TR/call-1.txt
  5. Update the JavaScript code in community-calls.html to fetch and display these files

For now, this is a standalone transcript page. In the community calls page, transcripts appear in a panel next to the video.
